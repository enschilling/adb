# Autonomous AI Lakehouse: Unlocking Enterprise Data for AI & Analytics

## About this Workshop

Welcome to the Kscope26 hands-on workshop for Oracle Autonomous AI Lakehouse. In this session, you will work with Oracle Autonomous AI Database tools that help teams bring enterprise data into a governed lakehouse experience for analytics and AI.

The workshop focuses on a compact event flow: prepare the workshop environment, load local data, explore AI-assisted table preparation, and use the Autonomous AI Database Excel Add-in. The presenter will demonstrate the AI feature portions of the Table AI Assist and Excel Add-in labs so attendees can see the complete experience without needing every AI capability enabled in each individual environment.

Estimated Time: 90 minutes

### Objectives

In this workshop, you will:

* Set up the workshop environment.
* Load local data files into Oracle Autonomous AI Database.
* Analyze and enrich table data with Table AI Assist.
* Install and use the Autonomous AI Database Excel Add-in.
* Observe presenter-led demos for AI-assisted features that may require preconfigured access.

### Workshop Flow

* **Lab 1:** Set up the Workshop Environment
* **Lab 2:** Load Local Data
* **Lab 3:** Analyze Data with Table AI Assist. AI feature steps are presenter-led.
* **Lab 4:** Analyze Data with the Excel Add-in. Attendees download and use the add-in; AI feature steps are presenter-led.

### Prerequisites

* An Oracle Cloud account or LiveLabs reservation for this workshop.
* Access to the Autonomous AI Database instance used in the labs.
* A supported browser for Oracle Cloud Console and Database Actions.
* Microsoft Excel for the Excel Add-in lab.

You may now proceed to the next lab.
